IjkPlayer 网络电台播放器 - 独立开发大纲
一、项目概述
开发一个 Android 网络电台 App，使用 Bilibili IjkPlayer 作为播放核心，支持：

电台列表展示（名称 + URL）

添加/删除电台

播放/暂停 MP3 网络流

音量调节

自动恢复上次播放的电台

播放状态提示（缓冲、播放、错误）

6.4 IjkPlayerManager.kt（核心）
单例模式，持有 IjkMediaPlayer 实例

初始化：加载 native 库，设置播放选项（硬解码、缓冲、重连）

实现 IMediaPlayer 的各种监听器，通过 LiveData 或 StateFlow 发送 PlaybackState

公开方法：playStation(station: Station)、pause()、resume()、stop()、setVolume(volume: Float)、isPlaying(): Boolean、release()

关键代码骨架：

kotlin
class IjkPlayerManager private constructor(context: Context) {
    private var ijkPlayer: IjkMediaPlayer? = null
    private val _state = MutableStateFlow<PlaybackState>(PlaybackState.Stopped)
    val state: StateFlow<PlaybackState> = _state.asStateFlow()

    init {
        IjkMediaPlayer.loadLibrariesOnce(null)
        IjkMediaPlayer.nativeProfileBegin("libijkplayer.so")
        initPlayer()
    }

    private fun initPlayer() {
        ijkPlayer = IjkMediaPlayer().apply {
            setOption(IjkMediaPlayer.OPT_CATEGORY_PLAYER, "mediacodec", 1)
            setOption(IjkMediaPlayer.OPT_CATEGORY_PLAYER, "opensles", 1)
            setOption(IjkMediaPlayer.OPT_CATEGORY_PLAYER, "start-on-prepared", 1)
            setOption(IjkMediaPlayer.OPT_CATEGORY_PLAYER, "reconnect", 5)
            setOption(IjkMediaPlayer.OPT_CATEGORY_PLAYER, "max-buffer-size", 1024 * 1024)

            setOnPreparedListener {
                start()
                _state.value = PlaybackState.Playing(currentStation?.name ?: "")
            }
            setOnErrorListener { _, what, extra ->
                _state.value = PlaybackState.Error("播放错误 [$what,$extra]")
                false
            }
            setOnCompletionListener {
                _state.value = PlaybackState.Stopped
            }
            setOnInfoListener { _, what, _ ->
                when (what) {
                    IMediaPlayer.MEDIA_INFO_BUFFERING_START -> _state.value = PlaybackState.Buffering
                    IMediaPlayer.MEDIA_INFO_BUFFERING_END -> {
                        if (isPlaying()) {
                            _state.value = PlaybackState.Playing(currentStation?.name ?: "")
                        }
                    }
                }
                true
            }
        }
    }

    fun playStation(station: Station) {
        currentStation = station
        ijkPlayer?.apply {
            dataSource = station.url
            prepareAsync()
        }
        _state.value = PlaybackState.Buffering
    }

    fun pause() { ijkPlayer?.pause() }
    fun resume() { ijkPlayer?.start() }
    fun stop() { ijkPlayer?.stop(); ijkPlayer?.reset(); currentStation = null; _state.value = PlaybackState.Stopped }
    fun isPlaying(): Boolean = ijkPlayer?.isPlaying() ?: false
    fun setVolume(volume: Float) { ijkPlayer?.setVolume(volume, volume) }
    fun release() { stop(); ijkPlayer?.release(); ijkPlayer = null; IjkMediaPlayer.nativeProfileEnd() }

    companion object {
        @Volatile private var INSTANCE: IjkPlayerManager? = null
        fun getInstance(context: Context): IjkPlayerManager = INSTANCE ?: synchronized(this) {
            INSTANCE ?: IjkPlayerManager(context.applicationContext).also { INSTANCE = it }
        }
    }
}

关键代码片段：

kotlin
override fun onCreate(savedInstanceState: Bundle?) {
    super.onCreate(savedInstanceState)
    binding = ActivityMainBinding.inflate(layoutInflater)
    setContentView(binding.root)

    storage = StationStorage(this)
    playerManager = IjkPlayerManager.getInstance(this)

    setupRecyclerView()
    setupButtons()
    observePlayerState()
    restoreLastPlayed()
}

private fun observePlayerState() {
    lifecycleScope.launch {
        playerManager.state.collect { state ->
            when (state) {
                is PlaybackState.Playing -> {
                    binding.buttonPlayPause.text = "停止"
                    binding.textStatus.text = "正在播放: ${state.stationName}"
                }
                is PlaybackState.Buffering -> binding.textStatus.text = "缓冲中..."
                is PlaybackState.Paused -> {
                    binding.buttonPlayPause.text = "播放"
                    binding.textStatus.text = "已暂停"
                }
                is PlaybackState.Stopped -> {
                    binding.buttonPlayPause.text = "播放"
                    binding.textStatus.text = "已停止"
                }
                is PlaybackState.Error -> {
                    binding.textStatus.text = "错误: ${state.message}"
                    Toast.makeText(this@MainActivity, state.message, Toast.LENGTH_SHORT).show()
                }
            }
        }
    }
}

八、编译与运行注意事项
SO 库架构：确保添加的 IjkPlayer 库与设备 CPU 架构一致。真机通常是 arm64-v8a，需将依赖改为 ijkplayer-arm64。如果同时包含多个架构可能导致包体积增大，但可接受。

网络权限：Android 9+ 默认禁止 HTTP 明文传输，必须在 manifest 中启用 usesCleartextTraffic="true"。

IjkPlayer 初始化：必须在播放前调用 IjkMediaPlayer.loadLibrariesOnce(null)，且在 Application 或首次获取单例时执行。

释放资源：onDestroy 中必须调用 release()，否则可能造成 native 内存泄漏。

九、调试与常见问题
问题	可能原因	解决方案
无法播放，logcat 无错误	硬解码不兼容	在 initPlayer 中将 mediacodec 设为 0 关闭硬解
播放几秒后无声	音频输出问题	禁用 opensles（设为 0），回退到 AudioTrack
缓冲时间过长	默认缓冲策略保守	减小 max-buffer-size 或增加 min-frames
切换电台崩溃	未正确 reset	调用 stop() 后再 reset()，再设置新 dataSource
后台播放被杀死	未使用前台 Service	需将播放器移至 ForegroundService 并添加通知
